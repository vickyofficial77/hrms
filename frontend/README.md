# Human Resource Management System (HRMS)

A modern Human Resource Management System developed using React, Tailwind CSS, Node.js, Express.js, and MySQL.

## Features

### Authentication

* User Registration
* User Login
* Session Management
* Logout

### Department Management

* Add Department
* Update Department
* Delete Department
* View Departments

### Position Management

* Add Position
* Update Position
* Delete Position
* View Positions

### Employee Management

* Add Employee
* Update Employee
* Delete Employee
* Search Employee
* Employee Status Tracking

### Reports

* Employees Currently On Leave
* Print Reports
* Export Reports

### UI Features

* Responsive Design
* Modern Dashboard
* Tailwind CSS Styling
* Toast Notifications
* Sidebar Navigation
* User Profile Display

## Installation

### Clone Repository

git clone https://github.com/vickyofficial77/hrms.git

### Backend Setup

cd backend

npm install

npm run dev

### Frontend Setup

cd frontend

npm install

npm run dev
